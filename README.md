# **RandomForest Model for Kaggle**  

### **What's this?**  
A simple RandomForest script I used for a Kaggle competition. Just a quick demo of my ML skills—no fancy production stuff here.  

### **Files**  
- `model.py` → Trains a RandomForest on the Kaggle data.  
- `/data` → Holds the dataset (from Kaggle).  

### **How to Run**  
1. Put your Kaggle data in `/data`.  
2. Run:  
```bash
python model.py
```  

### **What it does**  
- Loads the data  
- Basic preprocessing (fills missing values, encodes categories)  
- Trains a RandomForest  
- Saves predictions  
- Evaluate model metrics
